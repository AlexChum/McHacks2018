About SPACBot

Open source and integrated right into your team's CISCO Spark workspace.

SPACBot is a McHacks 2018 project. It has three main features.
It tells jokes, motivational quotes and chats about food.
The overall project is still ongoing, but someday, surely, SPACBot will be complete.
